## üéØ About

Club Ring Game is an innovative boxing game that combines:

- **Realistic Combat**: Fight Night Champion / Undisputed-level mechanics
- **Multi-Blockchain**: Simultaneous support for TON, Ethereum, Polygon, Solana, BSC
- **Play-to-Earn**: Earn real crypto through gameplay
- **Deep Progression**: 250+ achievements, 20-level career, 8 training types
- **Social Features**: Guilds, tournaments, PvP rankings

## üî• Quick Start

### For Players

1. Open Telegram or visit the web platform
2. Sign in with TON wallet (TON Connect 2)
3. Create your boxer and start training
4. Earn $RING tokens through fights and quests
5. Trade NFTs and stake tokens

### For Developers

```bash
# Clone repository
git clone https://github.com/kaylas000/Club-Ring-Game.git
cd Club-Ring-Game

# Install dependencies
npm install

# Set up environment
cp .env.example .env

# Start development
npm run dev
```

## üèÜ Game Features

### Combat System
- 60-second rounds with real-time mechanics
- 6-slot combo system (Jab, Cross, Hook, Uppercut, Guard, Slip)
- Stamina-based energy system (100 PEE per round)
- Health/damage mechanics with KO/TKO conditions

### Progression
- **Levels**: 20 career ranks (Amateur ‚Üí Legend)
- **Achievements**: 250+ challenges across 5 categories
- **Training**: 8 different training types
- **Tournaments**: Weekly, monthly, seasonal events

### NFT Collections
1. **Fighter NFT** - Your boxer with unique stats
2. **Land NFT** - Training centers with passive income
3. **Equipment NFT** - Gear with stat bonuses
4. **Badge NFT** - Achievement rewards (non-sellable)
5. **Booster NFT** - Temporary power-ups
